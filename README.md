## busta
Discord bot that will join and play their intro song

## Set Up
- `Python3` and `pip`
- Python Packages:
  - `discord`
  - `youtube-dl`
  - `pynacl`
  - `dotenv`
  - `flask`

## Environment setup
Use a `.env` file for discord tokens. The file should look like this

    TOKEN=mydiscordtoken
   

## Running
Run `py main.py` in root directory
